# MealBridge (Expo Go)

MealBridge is a mobile app concept for redistributing excess food from restaurants/hotels to NGOs.

## Core user roles
- **Restaurants/Hotels**: publish surplus food donations.
- **NGOs**: discover, reserve, and pick up available meals.

## Included functionality
- Role switcher for Restaurant and NGO experiences.
- Searchable live donations feed.
- Donation status visibility (Open, Reserved, Picked Up).
- Call-to-action adapts to selected role.

## Additional functionalities showcased
- Smart NGO-donation matching.
- Volunteer route/ETA tracking.
- Expiry and allergen safety tagging.
- Impact metrics (meals and CO₂ saved).
- In-app chat, pickup reminders, no-show alerts.
- Donation history and tax-ready reporting.
- Multi-language and accessibility support.

## Run locally
```bash
npm install
npm run start
```

Then scan the QR using **Expo Go**.
